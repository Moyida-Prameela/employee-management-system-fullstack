# employee-management-system-fullstack

## Backend
- Spring Boot
- MySQL
- JPA
- REST API

## Frontend
- React JS
- Bootstrap
- Axios

## Features
- Add Employee
- View Employees
- Update Employee
- Delete Employee

## Run Backend
cd EmployeeManagamentSystem-BackEnd
mvn spring-boot:run

## Run Frontend
cd ems-frontend
npm install
npm run dev
